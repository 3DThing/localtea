# Модуль Dashboard (Admin)

Агрегированная статистика для главной страницы админ-панели.

## Эндпоинты

### `GET /api/v1/dashboard/stats`
Получение сводной статистики.

**Возвращаемые данные**:

1.  **Orders**:
    *   `today`: Количество заказов за сегодня (с 00:00 UTC).
    *   `week`: Количество заказов за последние 7 дней.
2.  **Sales**:
    *   `today`: Сумма продаж (только оплаченные заказы) за сегодня.
    *   `week`: Сумма продаж за 7 дней.
3.  **Users**:
    *   `today`: Новые регистрации за сегодня.
    *   `week`: Новые регистрации за 7 дней.
4.  **Logs**:
    *   Список из 10 последних действий администраторов (Audit Log).
5.  **Sales Chart**:
    *   Массив данных для построения графика продаж за последние 30 дней.
    *   Формат: `[{date: "2023-10-01", amount: 15000}, ...]`.
