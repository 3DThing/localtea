# Модуль Orders (Admin)

Управление заказами клиентов.

## Эндпоинты

### `GET /api/v1/orders/`
Список заказов.
*   **Фильтры**: По статусу (`PENDING`, `PAID`, `SHIPPED`, `CANCELLED`).
*   **Пагинация**: `skip`, `limit`.

### `GET /api/v1/orders/{id}`
Детали заказа.
*   Возвращает информацию о заказе и список позиций (`items`) с привязанными SKU.

### `PATCH /api/v1/orders/{id}/status`
Изменение статуса заказа.
*   **Тело запроса**: `OrderStatusUpdate` (`status`).
*   **Логика**: Обновляет статус в БД. (В будущем: отправка email уведомлений).

### `POST /api/v1/orders/{id}/cancel`
Отмена заказа.
*   Устанавливает статус `CANCELLED`.
