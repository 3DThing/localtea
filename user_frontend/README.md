# LocalTea - Клиентский фронтенд

Современный интернет-магазин чая с тёмной фэнтезийной тематикой.

## Технологии

- **Next.js 16** — React фреймворк
- **Mantine 8** — UI компоненты
- **TanStack Query** — серверное состояние
- **Zustand** — клиентское состояние
- **Framer Motion** — анимации
- **TypeScript** — типизация

## Установка

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен сервера
npm start
```

## Переменные окружения

Создайте файл `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:8000/api/v1
```

## Структура проекта

```
src/
├── app/                    # Next.js App Router
│   ├── page.tsx           # Главная страница
│   ├── catalog/           # Каталог товаров
│   ├── blog/              # Блог
│   ├── login/             # Авторизация
│   ├── register/          # Регистрация
│   ├── cart/              # Корзина
│   ├── profile/           # Профиль пользователя
│   └── about/             # О нас
├── components/            # React компоненты
│   ├── layout/           # Header, Footer
│   ├── catalog/          # ProductCard и др.
│   └── blog/             # ArticleCard и др.
├── lib/                  # Утилиты
│   ├── api.ts           # API клиент
│   └── theme.ts         # Mantine тема
└── store/               # Zustand сторы
    └── index.ts         # Auth, Cart stores
```

## Функциональность

### Каталог
- Категории с баннерами
- Фильтрация и поиск
- Карточки товаров
- Детальная страница товара
- Выбор веса (SKU)
- Добавление в корзину

### Блог
- Лента статей
- Просмотры, лайки, комментарии
- Полнотекстовый поиск

### Пользователи
- Регистрация с подтверждением email
- Авторизация
- Профиль пользователя
- Корзина (для гостей и авторизованных)

### Взаимодействия
- Лайки для товаров и статей
- Комментарии
- Счётчики просмотров

## Docker

```bash
# Сборка образа
docker build -t localtea-frontend \
  --build-arg NEXT_PUBLIC_API_URL=https://api.localtea.ru/api/v1 .

# Запуск
docker run -p 3000:3000 localtea-frontend
```
